## Key gotchas discovered

- `govuk-!-background-colour-red` is NOT a valid GOV.UK Frontend utility class (P2-4). Was silently doing nothing. Fix: inline style or `.app-row--over-budget` in `app.css`.
- `govukServiceNavigation` nav items render as `<a>` tags (GET). The `/auth/logout` route is POST-only. So the "Sign out" header nav link is broken — clicking it returns 405. Working sign-out is the `<form method="post">` button. This is a known issue documented in CLAUDE.md.
